# Screenshots Directory

This directory contains screenshots and visual documentation for the GitAid project.

## Adding Screenshots

1. Add your screenshot images to this directory (`.png`, `.jpg`, or `.jpeg` format)
2. Reference them in `README.md` using the format:
   ```markdown
   ![Alt Text](/screenshots/filename.png)
   ```

## Recommended Screenshots

- `project-overview.png` - Main landing page
- `dashboard.png` - Dashboard view
- `ai-assistant.png` - AI Code Assistant interface
- `commit-intelligence.png` - Commit analysis view
- `team-collaboration.png` - Team features

## Image Guidelines

- Use PNG format for screenshots (better quality)
- Optimize images before committing (keep file sizes reasonable)
- Recommended dimensions: 1920x1080 or 2560x1440 for best quality
- Use descriptive filenames (kebab-case)
